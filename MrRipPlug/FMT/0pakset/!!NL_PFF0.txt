Идентификатор формата: PFF0

Заголовок формата:

struct {
  char id[4];     //Символы "PFF0"
  long DirOffset; //смещение начала FAT
  long DirSize;   //размер FAT в байтах
} PAK_Header;

Количество элементов FAT определяется по формуле:

nFiles = DirSize/64;

Описание элемента FAT:

Далее идет FAT.

struct {
  char Name[56]; //Имя файла
  long Offset;   //Размер файла
  long Size;     //Размер файла
} FATItem;

Такая структура повторяется nFiles раз и описывает один файл  из
WAD архива.

Смещение первого файла расчитывается по формуле:

Offset = nFiles*16+16;

Чтобы получить смещение остальных файлов то к Offset  надо  при-
бавлять Size предыдущего файла.

После идут непосредственно сами файлы.
